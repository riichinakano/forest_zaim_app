## 推奨事項

### 1. 科目マスタの保守方針

- **すべての科目を保持**: 使用頻度が低くても、過去データの参照に必要
- **新年度追加時の確認**: 新規科目コードが出現した場合は即座にマスタ追加
- **科目名変更の禁止**: 同一コードの科目名は絶対に変更しないこと

### 2. データ分析時の注意点

使用頻度の低い科目を分析する際は、以下に注意：

- **512, 514**: R3以降のデータには存在しない
- **542**: R5以降のデータには存在しない
- **611, 623**: 特定年度のみのデータ

### 3. 年度間比較時の注意

以下の科目は年度によって存在しない可能性があるため、欠損値処理が必要：

- 製造原価: 512, 514, 516, 520, 540, 542, 545, 553
- 販管費: 610, 611, 623, 630, 631, 632
