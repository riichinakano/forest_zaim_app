# 月次損益データフォルダ

このフォルダには、月次損益計算書のCSVファイルを配置します。

## ファイル配置方法

### ファイル名規則

```
{年度}_monthly.csv
```

例:
- `H27_monthly.csv` (平成27年度)
- `R6_monthly.csv` (令和6年度)

### CSVファイル形式

**エンコーディング:** Shift-JIS

**列構成:**
```csv
タイトル,科目コード,科目名称,当月迄累計金額,当月迄累計構成比,4月,5月,6月,7月,8月,9月,10月,11月,12月,1月,2月,3月
```

**データ例:**
```csv
タイトル,科目コード,科目名称,当月迄累計金額,当月迄累計構成比,4月,5月,6月,7月,8月,9月,10月,11月,12月,1月,2月,3月
損益計算書,410,売上高,12000000,100.0,1000000,1000000,1000000,1000000,1000000,1000000,1000000,1000000,1000000,1000000,1000000,1000000
損益計算書,620,役員報酬,3600000,30.0,300000,300000,300000,300000,300000,300000,300000,300000,300000,300000,300000,300000
```

## 注意事項

- **このフォルダ内のCSVファイルは`.gitignore`で除外されています**
- 内部情報を含むため、GitHubにはコミットされません
- 各環境で個別にデータファイルを配置してください

## 初期セットアップ

1. このフォルダに年度別のCSVファイルを配置
2. ファイル名が規則に従っているか確認
3. エンコーディングがShift-JISであることを確認
4. アプリを起動すると自動的に認識されます

```bash
streamlit run app.py
```
